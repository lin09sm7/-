## 🏆 亮點專案：社群媒體文案生成器

> 🏷️ **專案定位：** AI 驅動的社群內容產生平台，支援 YouTube／Facebook／Threads 三大平台，透過多角色人設自動生成不同風格的社群文案。

### 🛠️ 技術棧 (Tech Stack)
`Vite` | `Tailwind CSS` | `JavaScript` | `PocketBase` | `Cloudflare Workers` | `MaiAgent API`

### 💡 核心功能 (Key Features)

| 功能模組 | 細節說明 |
| :--- | :--- |
| 🤖 **多平台 AI 文案生成** | 上傳 PDF／TXT／MD 素材，由 AI 自動針對 YouTube、Facebook、Threads 產出平台專屬風格文案。 |
| 🎭 **多角色人設系統** | 每個平台內建 5 種預設人設（如竹科工程師、全能雙寶媽等），並支援自訂角色，產出多元觀點內容。 |
| 🔐 **安全 API 架構** | 透過 Cloudflare Workers 代理 API 請求，敏感金鑰不暴露於前端，搭配瀏覽器指紋驗證與 JWT Token 認證。 |
| 📚 **使用者管理與歷史紀錄** | PocketBase 後端支援註冊登入、生成歷史追蹤、自訂人設管理，資料依使用者隔離。 |

### 🚀 挑戰與突破 (Challenges Overcome)
> **挑戰：** 如何確保 AI 產出的文案格式一致且符合各平台特性，同時保護 API 金鑰安全？

**解決方案：**
設計嚴格的 Prompt 格式規範與驗證機制，確保輸出格式統一。透過 Cloudflare Workers 作為中間層代理 API 請求，將金鑰保留在伺服器端環境變數，前端完全無法接觸敏感資訊。
